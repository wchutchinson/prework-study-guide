HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The title element is important for SEO.
  • List items are nested in ordered or unordered blocks.
  • Tags represent the letters between < >.
  • Element is the content between the opening tag and closing tag.
  • <alt> are used to describe images and their inclusion can help webpage rankings for SEO.
